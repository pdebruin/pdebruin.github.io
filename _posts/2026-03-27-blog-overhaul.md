This blog needed some love. Over 200 posts, many without alt text or meta descriptions. Fixing that by hand would take days.

GitHub Copilot CLI made it a weekend project. I batched posts in groups of 15-30 and had Copilot generate descriptive alt text and meta descriptions across the entire archive. It also caught broken image references and fixed post content along the way.

On the content side: ~80 posts got alt text, ~150 got meta descriptions, and 5 LinkedIn posts I never published here got backfilled with images. Plus some SEO and config cleanup.

On the UI side: a full reskin using GitHub Copilot CLI's frontend design skill together with [color.adobe.com](https://color.adobe.com) for the palette. Hat tip to Burke Holland's video on GitHub CLI for getting me started: [YouTube](https://www.youtube.com/watch?v=CqcqWLv-sDM).
