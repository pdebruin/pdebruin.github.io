This Build was a great conference to attend remotely. Most shocking for many was the Azure (Mariner) Linux (https://github.com/microsoft/azurelinux) release. Something I would have never expected when I joined in 2011, but here we are. Interesting to see where this is going to go. 

Next, [azure developer cli went made generally available](https://devblogs.microsoft.com/azure-sdk/azd-ga-announcement/). I have mentioned azd before in this blog and love it to have simple tools that can help deploy a set of services. 

Another release, closer to my team, was about [Q&A assist](https://techcommunity.microsoft.com/t5/microsoft-learn-blog/microsoft-learn-announces-microsoft-q-amp-a-assist-and-new/ba-p/3614012), an addition to regular questions and answers forum to help find if the question has been asked before, so does it even need to be asked, and to help find answers in an automated fashion, avoiding to have to wait for an expert to look into yours.

All of the Build news was published in [book of news](https://news.microsoft.com/build-2023-book-of-news/)

Thanks for reading! :-)
